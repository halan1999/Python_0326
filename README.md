## ✍️ Quy Trình Làm Bài Tập (Best Practices)

Để chuyên nghiệp hóa quy trình làm việc của mình, hãy tuân thủ các bước:

1. **Cập nhật code mới nhất từ khóa học:** `git pull origin main`
2. **Tạo Branch riêng cho bài tập của bạn:** `git checkout -b [your-branch-name]`
3. **Thực hiện code:** Giải quyết các yêu cầu trong file bài tập.
4. **Commit và Gửi bài:**
   ```bash
   git add .
   git commit -m "feat: hoan thanh bai tap buoi X"
   git push origin [your-branch-name]
   ```
